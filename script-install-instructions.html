Installing Mob Commit Script

#!/usr/bin/env bash
USER_EMAIL=`git config --get user.email`
MOBSTERS=`cat ''`
# NOTE: authors must have an email to be valid
# so they need to be in the form "Some name, another name, ... <email@address.com>"
git commit -v --author="$MOBSTERS <$USER_EMAIL>"
  1. Copy this script into a file anywhere on your path.
  2. Name it anything you like (mob-commit, for example).
  3. Make sure it's executable with chmod +x mob-commit (or whatever you named your script).
  4. Instead of git commit, now run mob-commit!
Now your commit messages will look like this: 
             $ git log
commit 39d59e7e4c9acb021988b3040f9b7ace5f539b78
Author: James Kirk, Leonard McCoy, Spock 
Date:   Fri Mar 3 21:00:25 2017 -0500

    Set phasers to stun.
